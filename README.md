# 📖 Overview

This project is a **responsive promotional landing page** inspired by the **Kinder Joy Harry Potter Limited Edition** campaign.

The website delivers an immersive user experience through modern UI design, engaging visuals, smooth navigation, and responsive layouts. It demonstrates front-end development best practices using only **HTML5, CSS3, and JavaScript**.

The goal of this project is to recreate a premium product landing page that is visually appealing while maintaining clean code architecture and responsive performance across devices.
